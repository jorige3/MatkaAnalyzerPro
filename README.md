# Matka Analyzer Pro

This project provides a comprehensive analysis and prediction system for Matka games, incorporating various engines for frequency, cycles, digits, and momentum analysis. It also includes a confidence scoring system, a paper simulation for backtesting, and a modular architecture for easy expansion.

## Project Structure

- `main.py`: Main entry point for running the analysis.
- `data/`: Contains data files and schema definitions.
- `engines/`: Houses various analysis engines (Frequency, Cycles, Digits, Momentum).
- `scoring/`: Contains the Confidence Engine for combining results.
- `simulation/`: Includes the PaperSimulator for backtesting.
- `ui/`: (Future) User interface components.
